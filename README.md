# 🏦 Bank Loan Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing bank loan data using SQL and Power BI to provide insights into loan applications, funded amounts, repayments, customer profiles, and loan performance. The dashboard helps financial institutions monitor lending activities and make data-driven decisions.

---

## 🛠️ Tools & Technologies Used

- SQL
- Power BI
- DAX
- Data Modeling
- Data Visualization

---

## 📂 Dataset

The dataset contains information related to:

- Loan Applications
- Loan Status
- Funded Amount
- Loan Amount
- Interest Rate
- Debt-to-Income Ratio (DTI)
- Customer Information
- Loan Purpose
- Loan Term
- Home Ownership

---

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI
- Good Loan Percentage
- Bad Loan Percentage

---

### Visualizations

- Loan Applications Trend (Monthly)
- Funded Amount Analysis
- Loan Status Distribution
- Loan Purpose Analysis
- Home Ownership Analysis
- State-wise Loan Analysis
- Term-wise Loan Analysis
- Employee Length Analysis

---

## 📈 Business Insights

- Identify trends in loan applications.
- Monitor loan disbursement performance.
- Analyze repayment behavior.
- Evaluate loan quality through good and bad loan metrics.
- Understand customer demographics and borrowing patterns.
- Support risk assessment and lending decisions.
